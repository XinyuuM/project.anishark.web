# 流媒体平台与动画历史统计

aniShark 每季度进行一次统计、收集数据。

收集到的信息将非常有助于我们观察地缘政治、政策导向对ACG社区的影响。

> [!NOTE]
>
> 本统计数据源2017年1月至2025年7月由[H萌新番表](https://xf.hmacg.cn/)提供数据。
>
> 自2026年1月起，数据由自[XinyuuDB](https://db.xinyuu.cn/)提供。
