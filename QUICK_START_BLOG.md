## 📝 三步添加新文章

### 1️⃣ 创建文件

```bash
cd docs/blog/2026
touch 04-new-article.md
```

### 2️⃣ 写入内容

```markdown
---
title: 文章标题
description: 文章描述
date: 2026-04-01
---

# 文章标题

这里是文章内容...
```

### 3️⃣ 完成！

刷新浏览器，新文章自动出现。

❌ 避免中文文件名
